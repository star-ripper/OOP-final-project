# PetCo — Pet Store Inventory Management System

A Java-based inventory management system with a desktop GUI built for a pet store.

## Features
- Add and track pets with their individual characteristics
- Manage store products (food, toys, accessories) linked to pet types
- Object-oriented design using inheritance, encapsulation, and polymorphism
- Desktop GUI built with JavaFX for easy interaction

## Tech Stack
- Java
- JavaFX

## Context
Final project for Object-Oriented Programming course — COMSATS University Islamabad, Lahore Campus
